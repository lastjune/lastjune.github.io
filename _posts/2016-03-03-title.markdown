##代码

行内代码可以使用` `code` `的方式
多行使用做包围

可以在后跟语言表明代码的语言类型,使代码高亮

例如:

\`\`\`javascript

function(i){
    return i;
}

\`\`\`

```

function(i){
    return i;
}

```
